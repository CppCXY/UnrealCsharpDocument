### Publish

- PublishDirectory，DLL生成目录，默认为Script
- UEName，UE程序集名
- GameName，Game程序集名

### Bind

- BindClass，需要预先绑定的类型，可用于绑定CDO，比如UBlueprintFunctionLibrary

---

### Debug

- bEnableDebug，是否开启调试模式
- Host，调试地址
- Port，调试端口
